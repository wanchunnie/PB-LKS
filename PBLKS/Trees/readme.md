This folder saves visualized decision trees.

Run `python make_tree_plots.py` in terminal to create visualization of trees in our RandomForest model

## How can I interpret these trees?
Nodes in decision trees have following properties:
### criterion (Important):
Orgnized in form like `{feature_name} <= {threshold}` 

If the feature value satisfies the criterion in this sample, proceed to the left child node; if not, proceed to the right child node.

Leaf nodes does not have this property since no more splitting is needed any more.

### gini
The gini impurity of this node.

In decision trees, the Gini impurity of a particular node refers to the measure of purity or impurity at that node. It quantifies the probability of a randomly selected sample at that node being incorrectly classified into a different category.


### value
Orgnized in form like `value = [not_host_number, is_host_number]`. Indicating in our training dataset used to train this tree, `{not_host_number}` negative samples are left, `{is_host_number}` positive samples are left.

For instance, `value = [16, 5]` means when this node is reached, 16 negative samples(bacteria and phage interacts), 5 positive samples(does not interact) are left.

Traversing the tree, the samples are split and as a result, the value array reaching each node changes. 

### class
Class type of this node, provided in every node but only useful in leaf nodes.

This is actually decided by when reaching this node, which class has larger number of samples. The class of a leaf node indicates the prediction of this decision tree when it is reached in predicting.
